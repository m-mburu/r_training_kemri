  • Unknown sequence length (While loop) Sometimes you dont even know how long the input sequence should run for. This is common when doing simulations. For example, you might want to loop until you get three heads in a row. You cant do that sort of iteration with the for loop. Instead, you can use a while loop. A while loop is simpler than for loop because it only has two components, a condition and a body
  • mostly used in machine learning
output_while <- vector("double", ncol(df)) 
i =1
n_coldf = ncol(df)
while(i <= n_coldf){
  
   output_while[[i]] <- median(df[[i]])
   i = i+1 # decrementing function ensures that at the end the loop will terminate
   #print(i)
 
}

output_while
## [1] -0.3225057 -0.1180066 -0.7415445  0.1930928

Repeate Loop

  • Use when you want to repeat a calculation until a condition is reached

  • Mostly I recommend while loop

result <- 1
i <- 1
  
# test expression 
repeat {
  
   print(result)
     
   # update expression 
   i <- i + 1
   result = result + 1
  
   # Breaking condition
   if(i > 5) {
      break
   }
}

## [1] 2
## [1] 3
## [1] 4
